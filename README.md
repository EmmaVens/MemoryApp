# Трекер памяти

Простое Android-приложение на Java для проверки кратковременной памяти.

## Функции

- показ 5 случайных слов;
- таймер на запоминание;
- ввод запомненных слов;
- подсчёт результата;
- сохранение истории прохождений;
- очистка истории.

## Технологии

- Android Studio
- Java
- XML
- SharedPreferences
- CountDownTimer

## Как работает

Пользователь нажимает **«Начать тест»**, запоминает 5 слов, затем вводит их.  
Приложение считает количество правильных слов и выводит оценку.

## Оценка результата

- 5/5 — отличная память
- 3–4/5 — хороший результат
- 1–2/5 — средний результат
- 0/5 — низкий результат
